# Operator Logika
* Operator logika adalah operator untuk dua buah data ``` bool ```
* Hasil dari operator logika adalah ``` bool ``` lagi
* ``` && ``` -> Dan
* ``` || ``` -> Atau
* ``` ! ``` -> Kebalikan

# Operasi ``` && ```
| Nilai 1 | Operator | Nilai 2 | Hasil |
| ------- | -------- | ------- | ----- |
| `true` | `&&` | `true` | `true`  |
| `true` | `&&` | `false` | `false` |
| `false` | `&&` | `true` | `false` |
| `false` | `&&` | `false` | `false` |

# Operasi ``` || ```
| Nilai 1 | Operator | Nilai 2 | Hasil |
| ------- | -------- | ------- | ----- |
| `true` | `||` | `true` | `true` |
| `true` | `||` | `false` | `true` |
| `false` | `||` | `true` | `true` |
| `false` | `||` | `false` | `false` | 

# Operasi ``` ! ```
| Nilai 1 | Operator | Nilai 2 | Hasil |
| ------- | -------- | ------- | ----- |
| `!` | `true` | `false` |
| `!` | `false` | `true` |
